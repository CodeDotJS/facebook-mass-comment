# facebook-mass-comment

> comments 'n' times on any facbook post of a person or page.

## Installation

> sudo apt-get install python-pip

- sudo pip install httplib2
----
- pip install urllib3 [ install Certifi and PyOpenSSL ]
----
- pip install bs4
----
- pip install os
----
- pip install json
----
- pip install time
