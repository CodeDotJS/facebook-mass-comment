# facebook-mass-comment

> comments 'n' times on any facbook post of a person or page.

## Running 

> python facebook.py

## Modules needed :

__NOTE__ : Run the follwoing commands.

- pip install httplib2

- pip install urllib3 [ install Certifi and PyOpenSSL ]

- pip install bs4

- pip install os

- pip install json

- pip install time
